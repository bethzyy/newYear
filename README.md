# 马年新年红包小程序

## 项目简介

这是一个喜庆的马年新年主题微信小程序,以中国传统红包为设计灵感,点击后会弹出各种吉祥图案和包含"马"字的成语,营造浓厚的新年氛围。

## 功能特点

✨ **雅致中国风设计**
- 精美的红包样式界面
- 金红配色,喜庆大方
- 流畅的动画效果

🎯 **简单交互**
- 点击红包即可开始/停止动画
- 持续弹出吉祥元素
- 再次点击即可停止

🎊 **丰富内容**
- 马字成语:马到成功、一马当先、龙马精神等20个成语
- 吉祥图片:灯笼、金币、元宝、鞭炮、寿桃、鱼等
- 随机颜色和动画轨迹

## 使用方法

### 1. 安装微信开发者工具

下载并安装微信开发者工具: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择本项目目录 `C:\D\CAIE_tool\MyAIProduct\newYear`
4. 填写 AppID(可以选择测试号)
5. 点击"导入"

### 3. 添加图片资源

**重要**: 需要手动添加图片资源到 `images/` 目录:

- 从免费素材网站下载所需图片
- 参考 `images/README.md` 中的详细说明
- 确保图片格式为 PNG,背景透明
- 建议尺寸: horse.png (200x200px), 其他 (120x120px)

**临时方案**:
如暂无图片,可修改 `pages/index/index.js` 中的 `auspiciousImages` 数组,使用 emoji 替代:
```javascript
auspiciousImages: ['🏮', '💰', '🧨', '🍑', '🐟', '🎊']
```

### 4. 运行调试

1. 点击"编译"按钮
2. 在模拟器中预览效果
3. 点击红包按钮测试动画效果

## 技术实现

### 核心功能

1. **红包UI设计**
   - 使用 CSS 渐变和阴影实现立体效果
   - 金红配色体现中国风
   - 脉冲动画增加交互反馈

2. **动画系统**
   - 使用 `setInterval` 定时生成元素
   - CSS `@keyframes` 实现上升和旋转动画
   - 随机位置、颜色、时长增加趣味性

3. **内存管理**
   - 动画结束后自动清理元素
   - 页面卸载时清除定时器
   - 防止内存泄漏

### 自定义配置

可以在 `pages/index/index.js` 中修改:

- **成语列表**: 修改 `horseIdioms` 数组
- **图片资源**: 修改 `auspiciousImages` 数组
- **弹出频率**: 修改 `setInterval` 的时间(当前300ms)
- **动画时长**: 修改 `duration` 的随机范围(当前2-4秒)
- **颜色方案**: 修改 `colors` 数组
