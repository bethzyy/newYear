# 马年新年祝福 - 故宫风格大门

## 项目简介

这是一个**故宫风格的中国新春祝福应用**,采用传统建筑元素和现代技术相结合。点击故宫大门,会开启一段充满喜庆氛围的新年祝福之旅,包含新春音乐、烟花效果和28条马字成语。

## 功能特点

### 🏯 故宫风格大门
- **朱红大门**: 深红到暗红的渐变,故宫经典色彩
- **金色边框**: 8px粗金色边框,双层装饰线
- **门钉装饰**: 5行×9列=45个金色门钉,完全复原故宫大门
- **铜制门环**: 金色圆形门环,带有横向把手
- **3D开门动画**: 门向外旋转110度,真实的开门效果

### 🎵 新春音乐
- **五声音阶旋律**: C-D-E-G-C,中国传统音律
- **柔和音色**: 正弦波,音量0.1,不刺耳
- **自动播放**: 点击大门自动开始播放
- **循环播放**: 持续营造喜庆氛围
- **独立控制**: 右上角🎵按钮可暂停/播放

### 🎆 视觉效果
- **星光背景**: 50颗金色星星持续闪烁
- **烟花绽放**: 开门瞬间5个烟花,持续随机出现
- **闪光点缀**: 金色光点随机闪烁
- **成语祝福**: 28条马字成语从远到近弹出
- **发光标题**: "马年大吉"带有金色光晕

## 使用方法

### 方式一:手机版(最推荐!)

**特点**: 无需安装,直接在浏览器中运行

1. **电脑上预览**
   - 双击打开 `手机版.html`
   - 查看故宫大门和效果

2. **传到手机**
   - 用数据线/微信/QQ传输文件到手机
   - 用浏览器打开即可

3. **开始体验**
   - 点击故宫大门
   - 音乐响起,门打开
   - 享受成语祝福!

### 方式二:微信小程序版

**特点**: 可以发布到微信小程序平台

1. **下载工具**
   - 访问: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 下载并安装微信开发者工具

2. **导入项目**
   - 打开微信开发者工具
   - 选择"小程序"
   - 导入本项目目录
   - AppID选择"测试号"

3. **运行测试**
   - 点击"编译"按钮
   - 点击大门测试效果

## 技术实现

### 视觉设计
- **CSS 3D透视**: 真实的开门动画
- **渐变色彩**: 深红、金色、橙金
- **光晕效果**: text-shadow多层阴影
- **粒子系统**: 烟花和闪光效果

### 音频技术
- **Web Audio API**: 实时生成音乐,无需外部文件
- **五声音阶**: 中国传统音乐体系
- **增益控制**: 柔和的音量
- **循环播放**: 自动重复

### 交互设计
- **点击开门**: 主要交互方式
- **音乐控制**: 独立按钮控制
- **触摸支持**: 移动端优化
- **页面可见性**: 自动暂停/恢复
