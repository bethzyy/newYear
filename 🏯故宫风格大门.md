# 🏯 马年新年祝福 - 故宫风格大门

## ✨ 项目介绍

这是一个**故宫风格的中国新春祝福应用**,采用传统建筑元素和现代技术相结合,为您呈现雅致的新年庆祝体验。

---

## 🎨 设计特色

### 故宫风格大门
- **朱红大门**: 深红到暗红的渐变,故宫经典色彩
- **金色边框**: 8px粗金色边框,双层装饰线
- **门钉装饰**: 5行×9列=45个金色门钉,完全复原故宫大门
- **铜制门环**: 金色圆形门环,带有横向把手
- **3D开门动画**: 门向外旋转110度,真实的开门效果

### 新春音乐
- **右上角音乐按钮**: 🎵/🎶 切换图标
- **五声音阶旋律**: C-D-E-G-C,中国传统音律
- **柔和音色**: 正弦波,音量0.1,不刺耳
- **自动播放**: 点击大门自动开始播放
- **循环播放**: 持续营造喜庆氛围

### 视觉效果
- **星光背景**: 50颗金色星星持续闪烁
- **烟花绽放**: 开门瞬间5个烟花,持续随机出现
- **闪光点缀**: 金色光点随机闪烁
- **成语祝福**: 28条马字成语从远到近弹出
- **发光标题**: "马年大吉"带有金色光晕

---

## 🎵 音乐说明

使用Web Audio API实时生成音乐,无需外部文件。

### 音乐序列:
```
C5 → D5 → E5 → G5 → E5 → D5 → C5 → G4 → A4 → B4 → C5
(Do-Re-Mi-Sol-Mi-Re-Do-Sol-La-Si-Do)
```

典型的中国五声音阶旋律,喜庆而优雅!

### 音乐特点:
- **五声音阶**: 中国传统音乐体系
- **正弦波音色**: 柔和温暖
- **渐弱效果**: 每个音符自然衰减
- **音量适中**: 适合长时间聆听

## 🎯 使用方法

### 1. 打开应用
双击 `手机版.html` 文件

### 2. 观看大门
- 故宫风格的朱红大门
- 45个金色门钉装饰
- 铜制门环
- "点击开启"提示文字

### 3. 点击大门
- **音乐响起**: 柔和的新春旋律
- **门打开**: 3D旋转动画
- **烟花绽放**: 5个烟花同时绽放
- **显示祝福**: "恭喜发财"

### 4. 持续动画
- 28条成语从远到近弹出
- 随机闪光点缀
- 偶尔出现新烟花

### 5. 音乐控制
- 右上角🎵按钮
- 点击可暂停/播放
- 带有呼吸灯效果
