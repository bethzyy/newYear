# 图片资源说明

本目录需要放置以下新年主题图片资源:

## 必需图片列表:

1. **horse.png** - 马年吉祥物图标 (200x200px)
   - 用于红包按钮中心的马图案

2. **lantern.png** - 红灯笼 (120x120px)
   - 新年装饰元素

3. **coin.png** - 金币 (120x120px)
   - 财富象征

4. **ingot.png** - 金元宝 (120x120px)
   - 传统财富象征

5. **firecracker.png** - 鞭炮 (120x120px)
   - 新年喜庆元素

6. **peach.png** - 寿桃 (120x120px)
   - 长寿吉祥

7. **fish.png** - 鱼 (120x120px)
   - 年年有余

## 图片获取方式:

1. 可以从免费素材网站下载:
   - iconfont.cn (阿里巴巴图标库)
   - flaticon.com
   - freepik.com

2. 搜索关键词:
   - "马年 新年 中国风"
   - "Chinese New Year horse"
   - "新年喜庆 红包"

3. 图片要求:
   - PNG格式,透明背景
   - 尺寸如上所示
   - 色彩鲜艳喜庆(红、金、橙色为主)

## 临时方案:

如暂时没有图片资源,可以使用文字表情或纯色方块替代:
- 在 index.js 中修改 auspiciousImages 数组
- 使用 emoji 表情: 🏮 💰 🧨 🍑 🐟

## 注意:

请确保图片资源符合版权要求,建议使用原创或免费商用素材。
