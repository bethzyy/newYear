# 📱 手机版使用指南

## 🎉 好消息!现在可以直接在手机上运行了!

无需微信开发者工具,无需安装任何应用,直接用手机浏览器打开就能使用!

---

## 🚀 使用方法

### 方法一:电脑传到手机(推荐)

#### Windows系统:

1. **用数据线连接手机和电脑**

2. **传输文件**
   - 打开"此电脑"
   - 找到你的手机设备
   - 将 `手机版.html` 复制到手机

3. **在手机上打开**
   - 打开手机的"文件管理"应用
   - 找到 `手机版.html`
   - 点击打开
   - 选择用浏览器打开

#### 或者使用微信传输:

1. 在电脑上,用微信文件传输助手发送 `手机版.html`
2. 在手机微信中接收文件
3. 点击文件,选择用浏览器打开

---

### 方法二:部署到服务器(多人可访问)

如果你有自己的服务器或网站:

1. 将 `手机版.html` 上传到服务器
2. 生成访问链接(例如: https://你的网站.com/手机版.html)
3. 分享链接给朋友,大家都能访问!

---

### 方法三:使用本地网络(同一WiFi下)

1. **电脑端设置**:
   - 确保手机和电脑在同一WiFi
   - 使用Python启动简单服务器:
     ```bash
     # 进入项目目录
     cd C:\D\CAIE_tool\MyAIProduct\newYear

     # 启动服务器
     python -m http.server 8000
     ```

2. **手机端访问**:
   - 查看电脑的IP地址(在命令行输入 `ipconfig`)
   - 手机浏览器访问: `http://电脑IP:8000/手机版.html`
   - 例如: `http://192.168.1.100:8000/手机版.html`
