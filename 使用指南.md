# 马年新年红包小程序 - 使用指南

## 📥 第一步:下载微信开发者工具

### 方式一:官网直接下载(推荐)

1. **访问微信开发者工具官网**
   - 网址: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

2. **选择对应版本**
   - Windows系统:点击"Windows"下载按钮
   - macOS系统:点击"Mac"下载按钮
   - 文件大小约:150-200MB

3. **安装步骤**
   - 双击下载的安装包(例如:wechat_devtools_x.xx.exe)
   - 选择安装路径(默认即可)
   - 点击"安装"按钮
   - 等待安装完成(约1-2分钟)
   - 启动微信开发者工具

### 方式二:通过网盘下载

如果官网下载速度慢,可以从以下网盘下载:
- 百度网盘:搜索"微信开发者工具"分享链接
- 蓝奏云:https://www.lanzou.com/ 搜索"微信开发者工具"

## 🔑 第二步:注册微信小程序账号

### 如果已有账号,跳过此步骤

1. **注册页面**
   - 网址: https://mp.weixin.qq.com/
   - 点击"立即注册"

2. **选择账号类型**
   - 选择"小程序"

3. **填写信息**
   - 邮箱(未注册过微信的邮箱)
   - 密码
   - 确认密码
   - 验证码

4. **激活账号**
   - 登录邮箱点击激活链接
   - 填写主体信息(个人或企业)
   - **个人开发者**足够,不需要企业认证

5. **获取AppID**
   - 登录小程序后台
   - 开发 → 开发设置 → 开发者ID
   - 复制你的AppID(格式:wxXXXXXXXXXXXXXXXX)

### 使用测试号(快速体验,无需注册)

如果没有AppID,可以使用测试号:
- 打开微信开发者工具
- 选择"小程序"
- 点击"+"号
- 选择"测试号"
- 自动获得测试AppID

## 🚀 第三步:导入项目

### 1. 打开微信开发者工具

启动后会看到选择界面:
- 小程序
- 小游戏
- 公众号网页调试

**选择"小程序"**

### 2. 导入项目

点击"+"或"导入项目"按钮:

- **项目名称**: 马年新年红包
- **目录**: 点击"选择"找到本项目目录
  ```
  C:\D\CAIE_tool\MyAIProduct\newYear
  ```
- **AppID**:
  - 如果有注册账号:粘贴你的AppID
  - 如果没有:选择"测试号"或"游客模式"
- **开发模式**: 小程序
- **后端服务**: 不使用云服务

点击"导入"按钮

### 3. 首次加载

导入后会看到:
- ✅ 左侧:模拟器/编辑器
- ✅ 中间:代码编辑区
- ✅ 右侧:调试器

**如果遇到错误,请查看下方"常见问题"部分**

## 🎮 第四步:运行和测试

- 点击工具栏的"编译"按钮(或按 Ctrl+B)
- 模拟器会自动刷新显示效果

- 中间有"马年红包"文字

## 🔧 常见问题排查

### Q1: 导入项目后报错"图片找不到"

**原因**:图片路径不正确或文件不存在

**解决方案**:
- 确认 `images/` 目录下有所有SVG文件
- 检查文件名是否正确(必须是小写.svg)
- 刷新项目(重新编译)

### Q2: 点击红包没有反应

**可能原因**:
1. 代码未正确保存
2. 调试模式有错误

**解决方案**:
1. 点击"清缓存"→"清除文件缓存"
2. 重新编译
3. 查看调试器Console是否有报错

### Q3: 模拟器显示空白

**解决方案**:
1. 检查app.json中pages路径是否正确
2. 确保index.js的Page()没有语法错误
3. 重启微信开发者工具

### Q4: SVG图片不显示

**解决方案**:
- 微信小程序基础库需要 >= 2.3.0
- 在开发者工具设置中勾选"启用ES6转ES5"
- 尝试使用PNG格式替换SVG

### Q5: 动画卡顿或性能问题

**优化方案**:
- 减少`setInterval`的频率(修改index.js第52行的数值)
- 限制同时显示的元素数量
- 使用真机测试(模拟器性能较差)

### Q6: 无法在手机上预览

**检查清单**:
- 确保手机和电脑在同一网络
- 微信版本需要 >= 7.0
- 小程序开发权限已开启
- 重新生成预览二维码
