## 🎵 音乐说明:

### 使用Web Audio API生成音乐
- **不需要外部文件**: 纯代码生成
- **五声音阶**: C5-D5-E5-G5-E5-D5-C5-G4-A4-B4-C5
- **正弦波**: 柔和温暖的音色
- **渐弱效果**: 每个音符自然衰减
- **音量控制**: 只有0.1音量,非常柔和

### 音乐序列:
```
C5 → D5 → E5 → G5 → E5 → D5 → C5 → G4 → A4 → B4 → C5
(Do-Re-Mi-Sol-Mi-Re-Do-Sol-La-si-Do)
```

典型的中国五声音阶旋律,喜庆而优雅!
